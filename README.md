# fnct-news

This project scrapes news from the [Fukui National College of Technology](https://www.fukui-nct.ac.jp/) website and publishes the data as an open CSV dataset.

## Demo
- https://code4fukui.github.io/fnct-news/
- https://code4fukui.github.io/fnct-news/fnct-news.csv

## Features
- Fetches news data from the Fukui National College of Technology website
- Saves the data as a CSV open dataset
- Automatically updates the dataset on a schedule using GitHub Actions

## Requirements
This project uses Deno for the data fetching scripts. Deno v1.x is required to run the code.

## Usage
The CSV dataset is available at the demo links above. The fetching scripts can be run locally using Deno.

## Data / API
The project scrapes news data from the [Fukui National College of Technology news website](https://www.fukui-nct.ac.jp/news/).
